== Description ==

This Plugin Allow you to have Github Social Login in your site.

= Special Settings =

A section is added to your Settings menu, allowing you to configure your credentials and the styles for the button.

== Installation ==

1. Download MM Github Connect.
1. Upload the 'mm-github-connect' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
1. Activate MM Github Connect from your Plugins page.
1. Create a Github Application here <a href="https://github.com/settings/applications/new" target="_blank">https://github.com/settings/applications/new</a>.
1. Authorization callback URL needs to be your domain
1. Take the Client ID and Client Secret from your recent created App, and complete the fields in your settings.
